# 🃏 Blackjack Python

A simple, customizable Blackjack game written in Python.

This is a personal project created for fun, inspired by one of my favorite card games. The goal is to build a clean, modular, and easily configurable Blackjack implementation while keeping the codebase simple and maintainable.

## Features

* 🎮 Classic Blackjack gameplay
* 🌍 Multi-language support
* ⚙️ Configurable game settings through `config.ini`
* 🃏 Configurable number of decks
* 📜 Customizable Blackjack rules (via `config.ini`)
* 🧩 Modular project structure for easy maintenance

## Requirements

* Python 3.14 or newer is recommended.
* No external dependencies are required.

## Installation

Clone the repository:

```bash
git clone https://github.com/QuentinCHF/Blackjack_python.git
cd Blackjack_python
```

## Configuration

Most game settings can be customized directly from the `config.ini` file, including:

* Language
* Number of decks
* Blackjack rules

Simply edit the file before launching the game.

## Running the Game

```bash
python main.py
```
